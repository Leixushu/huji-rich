This folder contains various utilities and leftover projects created when developing the 3D Voronoi project.
All code should work with Visual Studio 2013, earlier, later or Linux compatibility is not guaranteed.

Old projects that are no longer used are kept in the Old subdirectory.
StressVoronoi is a Voronoi stress test application. 
TestVoronoi is a unit-test project utilizing gtest
PythonVoronoi is a Python version of the Voronoi calculation. It is not entirely functional and was created for debugging purposes.

Geometry3D is the solution that contains all these projects. It takes the actual Voronoi sources from the main source folder.
